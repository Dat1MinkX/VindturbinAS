# Utvikling-template
How-to use:
- [ ] `npm install`
- [ ] `npm update`
- `npm start`

# Database:
Bruk filen `initialize-database.js` for å gjøre endringer.

# Ruter
Ruter legges til i filen `pageRoutes.js`

# HTML, CSS og JavaScript
- IKKE endre på `base.css`
- Headers og footers kan lages i en felles fil - `header-footer.js`, dersom du ønsker det.
- DU SKAL ALDRI ENDRE I `app.js` THOMAS!!!
